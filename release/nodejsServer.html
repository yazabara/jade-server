Node Js

Чтобы ваш JavaScript код выполнился на вычислительной машине вне браузера (на backend), он должен быть интерпретирован и, конечно же, выполнен. Именно это и делает Node.js. Для этого он использует движок V8 VM от Google — ту же самую среду исполнения для JavaScript, которую использует браузер Google Chrome.

Как это работает?

Создаем проект

Создаем проект, определяем файлы Gruntfile.js, package.json. Создаем начальные директории для ресурсов

Gruntfile

Первый - Gruntfile.js или Gruntfile.coffee — файл конфигурации Гранта (грантфайл). Определеяет модули и таски.

package.json

Второй - package.json — описание проекта для npm. Содержит список зависимостей (в нашем случае это Грант и его плагины) и позволяет потом устанавливать их все одной командой.
