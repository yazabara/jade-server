Окружение

environment

Настройка окружения

Буквально вчера настраивал окружение на ноутбуке - и настройка заняла всего 30 минут, причем, большее время заняла загрузка , а не установка

Node js

Скачиваем nodeJs с официального сайта(http://nodejs.org/). Устанавливаем - если автоматически не добавиться переменная окружения - добавляем.

Ruby

Скачиваем Ruby (он нам пригодится для работы с SASS) с официального сайта так же ставим(http://rubyinstaller.org/).

SASS

Устанавливаем SASS(http://sass-lang.com/)

NPM (node package modules)

Устанавливаем необходимые npm модули(для конкретно нашего примера-сайта - есть bat-файл со всеми зависимостями)

ECHO Installing npm and plugins was started
npm install -g supervisor && npm install -g grunt-cli && npm install && npm install -g matchdep && npm install requirejs && npm install underscore && npm install async
ECHO Installing was finished
pause

IDE

Для разработки так же необходимо поставить какую-либо IDE, я рекомендую пользоваться intellij idea(webstorm).
